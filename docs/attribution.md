# Attribution

This repository is a portfolio and product-development artifact curated for
public release.

## Delivery Model

- Repository owner: Scott Caesar
- Delivery mode: human-directed, AI-assisted implementation and review
- Project shape: one orchestrated codebase spanning application code,
  infrastructure, operations scripts, security controls, and governance docs

Scott Caesar defined the scope, made product and architecture decisions,
reviewed outputs, curated the public release, and accepted the final delivered
state. Implementation and documentation work in this repository were produced
with AI assistance across multiple sessions and tools under that direction.

## How To Evaluate The Work

The best evidence is the repository itself:

- the current source tree
- ADRs and runbooks
- test and hardening scripts
- commit history for the public release lineage

This public repository is intentionally limited to materials that explain and
verify the software. Private client materials, commercial negotiation
documents, valuation memos, and internal diligence artifacts are not part of
the public release set.
