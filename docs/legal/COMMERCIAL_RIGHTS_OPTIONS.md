## Scope Truth (Read First)

Current repository state is a **hardened Phase 0/1 scaffold**, not a feature-complete secure file-sharing product.

Implemented value includes:
- infrastructure orchestration and deterministic bootstrap
- hardening controls and reproducibility checks
- backup/restore smoke and governance artifacts

Not yet implemented in runtime flows:
- auth/session lifecycle
- file ingest/download + encryption pipeline
- malware queue gate
- share policy runtime
- complete audit emission

## Option 1: Implemented Code Assignment (Snapshot Ownership)

### Typical Intent
- client wants full ownership of what is implemented now
- client runs roadmap independently

### Rights Shape
- assignment of implemented code for a pinned commit/snapshot
- ownership transfer limited to listed files/assets
- excluded by default unless listed: methodology docs, internal delivery playbooks, future improvements

### Price Range
- **$75,000 to $120,000** one-time

### Notes
- include transition window and handoff support boundaries
- include explicit warranty/maintenance limitations unless separately contracted

## Option 2: Hybrid License + Revenue Share + Collaborative Access (Recommended)

### Typical Intent
- client wants immediate operational use and contribution rights
- creator retains strategic IP position and supports quality/security governance

### Rights Shape
- client gets commercial license to implemented code snapshot
- client gets access to contribution documentation and governed workflow
- creator retains ownership of base IP unless later buyout/assignment is executed
- both parties can collaborate in shared repository under agreed controls

### Price Range
- upfront: **$30,000 to $55,000**
- revenue share: **5% to 12%** of defined commercialization revenue
- optional support retainer: **$2,500 to $8,000/month**

### Governance Terms To Define
- revenue basis (gross/net and allowed deductions)
- reporting cadence and payment terms
- audit and verification rights
- contribution/merge approval and security review gates
- optional buyout conversion path to full assignment

## Option 3: Full Partnership / Joint Venture

### Typical Intent
- both parties contribute materially to build, distribution, and operations

### Rights Shape
- jointly negotiated governance, economics, and decision rights
- may include equity plus reduced upfront fees

### Price Model
- custom; usually lower immediate cash and higher long-term shared upside/risk

### Notes
- use only when strategic alignment and operating expectations are clear

## Recommended Default Path

Start with **Option 2** for fairness and alignment:
- lower upfront burden for client
- protected creator upside
- continued technical stewardship and quality control
- clean future conversion to Option 1 if needed

## Commercial Trigger Statement (Reusable)

"Any commercialization, productization, revenue use, or use of this IP as a core enterprise asset requires a separate signed commercial rights agreement. Scope, ownership, and obligations are only those explicitly listed in signed schedules and commit-bound artifacts."
