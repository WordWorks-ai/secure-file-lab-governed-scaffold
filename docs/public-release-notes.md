# Public Release Notes

This repository is published as a portfolio-quality engineering artifact.

The public tree is intentionally limited to materials that help explain and
verify the software itself:

- source code
- infrastructure and operations scripts
- tests and quality gates
- ADRs, runbooks, and status artifacts

Non-public working materials are intentionally kept out of the public repo,
including:

- client-provided source materials
- valuation and pricing memos
- commercial negotiation templates
- consent and session paperwork

That curation is deliberate. The public repo is meant to show the engineering
work clearly without mixing it with private diligence or deal-room documents.
